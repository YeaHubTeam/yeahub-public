# YeaHub — платформа для подготовки к IT-собеседованиям, обучения и поиска работы

## Overview
YeaHub — open source платформа для подготовки к техническим собеседованиям в IT, обучения разработчиков, карьерного роста и поиска работы.

Сервис объединяет базу вопросов с реальных собеседований, livecoding задачи, аналитику по компаниям, тренажёр для запоминания, а также инструменты для поиска работы и оптимизации резюме под ATS (Applicant Tracking Systems).

Подходит для подготовки к собеседованиям и трудоустройства в IT-компании (Сбер, Яндекс, VK, Ozon и другие).

## Key Features

### 1. Interview Questions (вопросы с собеседований)
- 10000+ вопросов с реальных технических интервью
- Алгоритмы, структуры данных, system design, языки программирования
- Разделение по навыкам, ролям и уровням
- Ответы с объяснениями
- Поиск и фильтрация

**Use case:** подготовка к техническому собеседованию разработчика

### 2. Livecoding (задачи для coding interview)

- Поддержка: JavaScript, Python, Java, Go, SQL

**Use case:** практика алгоритмов и livecoding интервью
